#### emoji 插件用法

显示表情图标
```markdown
:emojikey:
```

禁用解析成表情
```markdown
_colon_;emojikey:
或
&#58;emojikey:
```

<br>

#### 忽略某张图片缩放
```markdown
![](image.png ":no-zoom")
```
